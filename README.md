# 📊 Student Spending Dataset Dashboard (MS Excel)

## 📌 Overview

This project analyzes student spending patterns using Microsoft Excel. The dashboard provides interactive visualizations that help understand how students spend money across different categories and identify key spending trends.

The project demonstrates Excel skills including data cleaning, Pivot Tables, Pivot Charts, Slicers, KPI calculations, and dashboard design.

---

## 📂 Dataset

The dataset contains student spending information such as:

- Student details
- Spending categories
- Amount spent
- Payment method
- Age
- Gender
- Other spending-related attributes

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Excel Formulas

---

## 📈 Dashboard Features

The interactive dashboard includes:

- Total Spending
- Average Spending
- Spending by Category
- Spending by Gender
- Spending by Age Group
- Payment Method Analysis

---

## 📊 Key Insights

- Identified the highest spending categories.
- Compared spending habits across genders.
- Analyzed spending trends based on age groups.
- Evaluated preferred payment methods.
- Created interactive filters for dynamic analysis.
